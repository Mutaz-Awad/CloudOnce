Checks for duplicate achievement- or leaderboard IDs. If a duplicate is found it will draw an EditorGUILayout.HelpBox informing the user where the duplicate was detected.

Parameters
platformIdListA list of either achievements or leaderboards.
checkAppleIDsCheck for duplicates in the Apple IDs.
checkGoogleIDsCheck for duplicates in the Google IDs.
listNameShould be either "achievement" or "leaderboard". Is included in the warning message.
Returns
Return true if no duplicates are found, false if at least one duplicate exists.
